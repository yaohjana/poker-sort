📚 遊戲說明

  • 選擇排序方式：從下拉選單選擇想要排序的規則（數字、花色、顏色等）。
  • 發牌：點擊「重新發牌」後，系統會隨機產生卡片排列。
  • 排序：點擊兩張卡片即可交換位置（點擊第一張會標記為選擇狀態）。
  • 檢查：完成後點擊「檢查答案」看是否正確。
  • 提示：如果卡住，可以點擊「顯示提示」或打開完成提示勾勾協助思考。
  • 統計：上方統計欄會記錄你的比較次數與交換次數。
  • 重新發牌 vs 重置本局：「重新發牌」會取得新的一組牌；「重置本局」則恢復此次發牌的原有模樣，可重複練習同一組牌。

💡 排序演算法小知識

選擇排序方式後，這裡會顯示相關的排序演算法概念。

提示：先從少量牌開始，感受「比較 → 交換」的節奏。
  1. 在上方選擇一種排序目標（例如：數字大小）。
  2. 觀察目前牌的排列，想像「理想順序」應該長什麼樣子。
  3. 點擊一張牌，再點擊另一張牌，完成一次交換。
  4. 重複比較與交換，直到你覺得已經排好為止，再按「檢查答案」。

📋 常見排序法：使用情境與複雜度

選擇排序（找極值）

使用情境

每輪從未排序區找最小（或最大），與當前位置交換。本程式：原地模式可「選一張與目標位置交換」；額外空間模式可從上方取一張放到下方最後一格。

時間複雜度

  • 最好 O(n)
  • 平均 O(n²)
  • 最壞 O(n²)

空間複雜度

O(1) 原地

插入排序（插對位置）

使用情境

每輪取未排序區一筆，插入已排序區的「正確位置」。本程式：額外空間模式，選上方一張牌，再點下方虛線格插入正確位置。

時間複雜度

  • 最好 O(n)
  • 平均 O(n²)
  • 最壞 O(n²)

空間複雜度

O(1) 原地

氣泡排序（相鄰交換）

使用情境

重複掃描，比較「相鄰」兩筆，順序錯就交換。本程式：原地模式，只交換相鄰的兩張牌，一輪輪把較大者往右推。

時間複雜度

  • 最好 O(n)
  • 平均 O(n²)
  • 最壞 O(n²)

空間複雜度

O(1) 原地

快速排序（分治法）

使用情境

選一筆當 pivot，小於的放左、大於的放右，再對左右兩邊遞迴。本程式：可先理解分治概念；實際分區操作較複雜，以說明與複雜度認識為主。

時間複雜度

  • 最好 O(n log n)
  • 平均 O(n log n)
  • 最壞 O(n²)

空間複雜度

O(log n) 遞迴堆疊
