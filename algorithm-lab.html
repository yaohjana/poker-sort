🌟 認識演算法（獨立重點）

演算法就是：把一個問題，拆成「可照做」的步驟。
最基本結構是 輸入 → 步驟 → 輸出，而且要有停止條件。

輸入 Input：現在有哪些資料 步驟 Process：每一步怎麼做 輸出 Output：最後得到什麼

☔ 生活例 1：出門要不要帶傘

查看天氣預報→ 判斷降雨機率→ 高於 50% 帶傘→ 完成出門準備

🍱 生活例 2：便當店點餐流程

看菜單→ 選主菜與配菜→ 確認金額→ 付款取餐

📚 生活例 3：圖書館借書

查詢書籍→ 找到書架位置→ 借書機掃描→ 完成借閱

🔍 邏輯判斷演算法：顯微鏡定位

重點：判斷（if / else）也是演算法流程的一部分。大問題（把影像移回中央）先拆成小問題，再用條件判斷決定流程方向。

Step 1：左右判斷

影像在右邊 → 玻片往右移
影像在左邊 → 玻片往左移
影像在中線 → 左右不移動

Step 2：上下判斷

影像在上方 → 玻片往上移
影像在下方 → 玻片往下移
影像在中線 → 上下不移動
開始：影像位置（輸入）
if 影像在右邊 then 玻片往右；else if 在左邊 then 玻片往左；else 不移動
if 影像在上方 then 玻片往上；else if 在下方 then 玻片往下；else 不移動
輸出：合併後的移動方向（例如右上方）

互動演練（判斷分支決策）

影像出現在「右上方」，請先做左右判斷，再做上下判斷。

A. 左右判斷（if / else）
B. 上下判斷（if / else）
先用 if/else 完成左右，再用 if/else 完成上下；最後合併方向。
例：影像在右上方 →（左右=右）+（上下=上）→ 玻片往右上方移動。
